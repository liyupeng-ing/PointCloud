Detailed Description

This class implements the clustering algorithm.

This includes:

  • A fast pre-clustering step intended to speed up the actual clustering.
  • Density calculation.
  • Full clustering.
  • Outlier removal.

See Cloud Point Analysis for detailed documentation of the underlying algorithms.
