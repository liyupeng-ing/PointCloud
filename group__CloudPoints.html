Main function.

This program performs the following tasks:

  • Parse the command line and defines configuration.
  • Read data from a text file.
  • Runs a spacial clustering to identify players on the field.
  • Runs a color analysis to identify the teams.

Full documentation of the algorithms is available in Cloud Point Analysis.

Configuration details:

  • -v, --verbose Turns ON verbose mode.
  • -i, --inputFile Name of the data input file.
  • -f, --evaluationDataFraction Fraction of data to use for evaluation.
  • -p, --skipPreClustering Don't run pre-clustering.
  • -P, --preClusteringSize Size parameter in unit length for pre-clustering.
  • -d, --densityWindow Size of the window used to compute densities.
  • -D, --seedDensityThreshold Density threshold for seed selection, normalized to maximum density.
  • -c, --clusterCoreSize Size parameter in units of standard deviations for outlier removal.
  • -u, --unsupervisedClassification Run unsuppervised classification.
  • -l, --nLayersPerCluster Number of layers per cluster for color analysis.
  • -t, --runMVATraining Runs MVA training when runing in supervised classification mode.
  • -o, --tmvaOutputFile Output file to save TMVA performance histograms. Put "None" to skip saving histograms.
  • -T, --truePositionsFileName File name containing truth player positions and teams.
  • -N, --trainingClustersSplitN Number of sub-clusters for training splitting.
  • -F, --trainingClustersSplitF Fraction of points in each sub-cluster for training splitting.
  • -K, --maxKmeansIterations Maximum number of k-means iterations during PCA/kmeans classification.
