bool DataSet::readFromFile ( const Config &  config,
DataSet &  trainingData,
DataSet &  evaluationData 
)
static

Reads data from a text file.

Expected format: Each line of the input file has 6 columns (X, Y, Z, R, G, B)

  • X, Y, Z: the ​ xyz position of the 3d point
  • R, G, B: the ​ rgb component of the color of the 3d point

Data points are split into training and evaluation sets. The ranges (min,max) of the data coordinates is comuted at this stage.

true upon success, false upon failure.
